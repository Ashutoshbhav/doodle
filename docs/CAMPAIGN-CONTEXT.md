## 8. Landing page plan (`/drop`)

- Dedicated single-CTA route in the existing Next.js app; no nav; brand-locked; message-matched to the chosen creative direction (pending §7.1).
- One conversion: **email → waitlist** (Neon Postgres, chosen).
- Attribution: reads `utm_source` (meta/google) + `utm_content` (creative id) → stored in `waitlist.source` as `lcl_may26:<src>:<creative>` → **per-creative conversions + CVR for the deck** (the strongest "insight" section).
- Tracking: Meta Pixel `Lead` + Google Ads conversion, env-gated, dedup via the existing server action. PostHog already wired as backstop.
- `robots: noindex` (paid LP, no organic/duplicate-content footprint).
- UTM scheme: `utm_campaign=doodle_lcl_may26`, `utm_source=meta|google`, `utm_medium=cpc`, `utm_content=<creative id>`.
